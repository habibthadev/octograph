## Quick Start

### Installation

```bash
npm install octograph
# or
pnpm add octograph
# or
yarn add octograph
```

### Basic Usage

```tsx
import { OctoGraph } from "octograph";
import "octograph/styles";

function App() {
  return <OctoGraph username="habibthadev" year={2023} theme="github" />;
}
```

### With React Query Provider

```tsx
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import { OctoGraph } from "octograph";

const queryClient = new QueryClient();

function App() {
  return (
    <QueryClientProvider client={queryClient}>
      <OctoGraph username="habibthadev" />
    </QueryClientProvider>
  );
}
```

## GitHub Token Setup

For production use, you'll need a GitHub personal access token:

1. Go to [GitHub Settings > Developer settings > Personal access tokens](https://github.com/settings/tokens)
2. Generate a new token with `read:user` scope
3. Add it as `GITHUB_TOKEN` or `NEXT_PUBLIC_GITHUB_TOKEN` environment variable

```bash
# .env.local
GITHUB_TOKEN=your_token_here
```
